# 4wise Landing

Bu repoda iki uygulama var:

- **Next.js (Vercel için önerilen)**: `next/`
- **Expo (eski prototip)**: repo kökü

## Next.js (Vercel)

```bash
cd next
npm install
npm run dev
```

### Vercel Deploy

- **Root Directory**: `next`
- Vercel otomatik olarak `next build` / `next start` akışını kullanır.

## Expo (opsiyonel)

Expo tarafı Vercel’da “en rahat” seçenek değildir; Next.js sürümünü kullanın.
